# Shorts Clipper — Автоматическая нарезка Shorts/Клипов

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Десктопное веб-приложение для загрузки длинных видео, автоматической нарезки на вертикальные Shorts/Клипы с субтитрами, баннерами и публикацией в YouTube/VK.

**Architecture:** FastAPI backend + Vanilla JS frontend (SPA). SQLite для хранения метаданных. MoviePy/FFmpeg для видеообработки. faster-whisper для распознавания речи (CPU/GPU).

**Tech Stack:** Python 3.11, FastAPI, SQLAlchemy+aiosqlite, MoviePy, FFmpeg, faster-whisper, Jinja2 (для шаблонов), Vanilla JS + Tailwind CSS (CDN).
